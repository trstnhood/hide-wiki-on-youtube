# Hide Wiki – Google Chrome

## Was macht dieses Plugin?

Dieses Plugin entfernt die Wikipedia- und Britannica-Infoboxen, die YouTube automatisch unter bestimmten Videos einblendet. Keine Tracking-Daten, keine Werbung, kein Quatsch – es blendet einfach nur diese Boxen aus.

---

## Installation

> **Kurzer Hinweis:** Chrome erlaubt die Installation von Plugins außerhalb des Chrome Web Store leider nur über den „Entwicklermodus". Das klingt technisch, ist aber in 30 Sekunden erledigt.

### Schritt 1: ZIP entpacken

Im Hauptordner liegt die Datei:

```
hide-wiki-chrome.zip
```

**Diese Datei entpacken:**
- **Mac:** Doppelklick auf die ZIP-Datei
- **Windows:** Rechtsklick → „Alle extrahieren…"
- **Linux:** Rechtsklick → „Hier entpacken"

Es entsteht ein Ordner mit drei Dateien (`manifest.json`, `content.js`, `content.css`).

### Schritt 2: In Chrome installieren

1. **Chrome öffnen**
2. In die Adressleiste eingeben: `chrome://extensions` und Enter drücken
3. **Oben rechts** den Schalter **„Entwicklermodus"** einschalten
4. Auf **„Entpackte Erweiterung laden"** klicken (erscheint oben links)
5. **Den entpackten Ordner auswählen**
6. **Fertig!**

> Chrome zeigt gelegentlich beim Start einen Hinweis zu Erweiterungen im Entwicklermodus. Einfach auf **„Erweiterungen behalten"** klicken – das Plugin funktioniert ganz normal.

---

## Auch für diese Browser

Diese Anleitung funktioniert genauso für:
- **Arc** (nutzt `chrome://extensions`)
- **Brave** (nutzt `brave://extensions`)
- **Vivaldi** (nutzt `vivaldi://extensions`)
- **Opera** (nutzt `opera://extensions`)

Einfach die jeweilige Adresse verwenden – der Rest ist identisch.

---

## Funktioniert es?

1. Geh auf YouTube
2. Suche nach einem Thema mit Wikipedia-Box (z. B. „COVID-19" oder „Klimawandel")
3. Die Infobox unter dem Video sollte nicht mehr sichtbar sein

## Deinstallation

1. `chrome://extensions` aufrufen
2. „Hide Wiki" finden
3. Auf **„Entfernen"** klicken
